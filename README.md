# Sales Data Analysis Dashboard

## Overview
This project is an interactive Sales Data Analysis Dashboard built using Streamlit.

## Features
- Total Sales
- Total Profit
- Total Orders
- Sales by Category
- Profit by Region
- Sales Trend
- Interactive Filters
- Sales Data Table

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly

## Run

```bash
streamlit run app.py
```

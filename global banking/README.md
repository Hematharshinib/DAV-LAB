# Global Banking Data Analytics & Visualization

## Overview
This project is an interactive dashboard for analyzing banking data from multiple countries.

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly

## Features
- Dashboard with KPI Cards
- Country Filter
- Account Type Filter
- Balance by Country
- Account Type Distribution
- Loan Amount Analysis
- Credit Score Analysis
- Download Filtered Data

## Run the Project

```bash
python -m streamlit run app.py
```
